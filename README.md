# AI-Powered-Resume-Analyzer
AI-powered Resume Analyzer SaaS that evaluates resumes using ATS scoring, analyzes skills and content quality, and provides smart feedback. Built with MERN stack and OpenAI API, it features real-time processing and a modern UI to help users optimize resumes for software roles.
# Resume Analyzer 🚀

AI-powered ATS Resume Analyzer built with:

- React
- Node.js
- MongoDB
- OpenAI API

## Features
- ATS Score
- AI Feedback
- Resume Upload
- Dashboard UI

## Run Project

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
